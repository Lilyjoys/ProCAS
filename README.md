# ProCAS  
**Multi-Lysine Post-Translational Modification Site Prediction Based on the SPUS Algorithm and the ProCAS Architecture**

---

## Installation and Usage

### Clone the Repository
```bash
git clone https://github.com/Lilyjoys/ProCAS.git
cd ProCAS
```
## Dependencies

Please ensure the following packages are installed before running the
code:

``` text
numpy
torch
scikit-learn
propy3
pickle
```


### Run Prediction
```bash
python main.py 
```
## Model Training

If you wish to **train the model from scratch**, run:

``` bash
python train.py
```

The training pipeline integrates feature extraction, class imbalance
handling, and multi-label performance evaluation.
